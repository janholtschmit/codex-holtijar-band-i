# CODEX HOLTJAR – Band I

A Documentary Case Study in Lineage, Authority, and Mobility Across the Rhine–Westphalian Corridor

Author: Jan Holtschmit  
Year: 2026

---

## Description

This work presents a document-based reconstruction of the occurrence of the name **Holtschmit / Holtschmidt** across multiple functional contexts between the 15th and 20th century.

The study is based exclusively on archival sources and does not attempt to establish genealogical continuity.

---

## Contents

- Unna Urkunden (15th century administrative records)
- Balve Akten (17th century legal and economic records)
- Administrative and industrial references (19th century)
- Maritime documentation (20th century)
